# AI / LLM 项目

收录 AI、LLM、Agent、RAG、模型应用、推理部署等相关 GitHub 项目。

## 收录列表

## [hugohe3/ppt-master](https://github.com/hugohe3/ppt-master)

- 类型：GitHub 项目 / AI Agent Skill
- 分类：AI / LLM、PPT 生成、内容生产
- 标签：PPTX, PowerPoint, AI Agent, Claude Code, Codex, Cursor
- 推荐理由：可以从 PDF、DOCX、URL、Markdown 等材料生成真实可编辑的 PowerPoint，不是把页面截图塞进 PPT，适合学习 AI 生成办公文档的工作流设计。
- 适合场景：资料转演示稿、AI IDE 内生成 PPT、研究 Agent Skill 的提示词与脚本组织方式。
- 备注：偏“原生可编辑 PPTX”路线，和生成 HTML 演示稿的工具可以互补。

## [op7418/guizang-ppt-skill](https://github.com/op7418/guizang-ppt-skill)

- 类型：GitHub 项目 / AI Agent Skill
- 分类：AI / LLM、演示文稿、视觉生成
- 标签：PPT, HTML Slides, Claude Code, Codex, Skill, Editorial Design, Swiss Design
- 推荐理由：面向 AI Agent 的演示稿生成 skill，重点在杂志感、瑞士风格、封面和配图提示等视觉系统，适合参考它如何约束 Agent 输出质量。
- 适合场景：文章转演示、内容型 PPT、单文件 HTML 演示稿、AI 视觉排版工作流。
- 备注：交付形态偏 HTML 演示，与 `ppt-master` 的 PPTX 路线不同。

## [z2z23n0/agent-capsule](https://github.com/z2z23n0/agent-capsule)

- 类型：GitHub 项目 / CLI / Agent 工具
- 分类：AI / LLM、Agent 协作、会话迁移
- 标签：Codex, Claude Code, Session Sharing, Handoff, CLI, Cloudflare Workers
- 推荐理由：把完整的 Codex 或 Claude Code 会话打包成可分享链接或 capsule 文件，接收方可以导入到本地继续工作，很适合研究 Agent 协作和上下文交接。
- 适合场景：复杂任务交接、会话归档、跨机器恢复 Agent 上下文、Agent 工具链研究。
- 备注：会话可能包含敏感路径、工具输出和截图，分享前要检查隐私内容。
